button控件效果展示
属性 
        content：内容展示
        render：目标对象
        width：控件宽度
        height：控件高度
        elCls ： 控件样式
        tooltip：鼠标悬停显示的文字
事件 
        button1.on("click",function(){alert(1)});
        使用方式与普通dom元素事件绑定类似。
button控件代码展示 
        require(["button","button/assets/dpl.css"],function(Button){
            var button1 = new Button({
                content : 'ButtonContent',
                render : '#container',
                width : '150px',
                height : "30px",
                elCls : 'ks-button-success',
                tooltip : "11"
            });
            button1.on("click",function(){alert(1)});
            button1.render();
        });
